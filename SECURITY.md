# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in KoreShelf, please **do not** report it in the public Issue tracker. 

Instead, please report it via email to:
**[koreshelf@gmail.com]**

### What to include in your report:
* A description of the security issue.
* Steps to reproduce the vulnerability.
* Any code snippets or screenshots that might help in understanding the issue.

### Our Commitment
We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible. Please allow us reasonable time to investigate and fix the problem before making any information public.

Thank you for helping keep KoreShelf safe for everyone!
